# VoteSecure - Online Voting System

## Overview
VoteSecure is a secure, modern, and user-friendly online voting platform designed to facilitate democratic elections with transparency and security.

## Features
- **User Authentication**: Separate login systems for voters and administrators
- **Voter Registration**: Easy registration process with CNIC verification
- **Multiple Elections**: Support for multiple concurrent elections
- **Real-time Results**: Live election results and statistics
- **Admin Dashboard**: Comprehensive election and candidate management
- **FAQ Section**: Helpful information for voters
- **Contact Support**: Easy communication with support team

## Pages
- **Home**: Landing page with statistics and call-to-action
- **About**: Information about the platform
- **FAQ**: Frequently asked questions
- **Contact**: Contact form and support information
- **Voter Login/Register**: Authentication for voters
- **Admin Login**: Secure admin access
- **Voter Dashboard**: View and participate in elections
- **Admin Dashboard**: Manage elections and candidates
- **Voting Interface**: Cast votes for elections
- **Results**: View election results and statistics

## Design
- Dark theme with blue and purple gradient accents
- Responsive design for all devices
- Smooth animations and transitions
- Modern UI with Lucide icons

## Technology Stack
- HTML5
- CSS3 with custom variables
- Vanilla JavaScript
- Lucide Icons
